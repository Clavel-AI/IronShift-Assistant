### Response Time
We aim to respond within 48 hours and will keep you updated on the remediation progress.

## Security Best Practices

When using IronShift:

1. **Review all actions** before approving them
2. **Keep CLI tools updated** (Azure CLI, AWS CLI)
3. **Use principle of least privilege** for cloud credentials
4. **Audit conversation history** regularly
5. **Self-host backend** for sensitive environments
